# Исходные данные

Эта директория содержит **placeholder** для исходных данных, которые используются для генерации dashboard.

⚠️ **ВАЖНО**: Исходные данные НЕ включены в репозиторий из-за большого размера и лицензионных ограничений.

## Данные для воспроизведения

Если вы хотите воспроизвести pipeline полностью (Вариант Б), вам понадобятся следующие файлы:

### 1. СберИндекс POAD

**Файл:** `Data_SberIndex_POAD.xlsx` (~500 KB)

**Содержание:**
- 93 показателя привлекательности (POAD)
- 128 населённых пунктов
- Метрики социально-экономического развития

**Источник:** СберИндекс (https://www.sberindex.ru/)

**Лицензия:** CC BY-SA 4.0

**Где получить:**
- Официальный сайт СберИндекс
- Или использовать готовый dashboard в `/dashboard/dashboard.html` (данные уже встроены)

### 2. Потребительские расходы

**Файл:** `consumption.parquet` (~2 MB)

**Содержание:**
- 303,000+ записей потребительских расходов
- 2,571 территория
- Данные за 2022-2023 годы

**Источник:** СберИндекс API

**Формат:**
```
territory_id | year_month | category | amount | ...
```

### 3. Доступность рынков

**Файл:** `market_access.parquet` (~1.5 MB)

**Содержание:**
- Индексы доступности рынков
- Расстояния до крупных городов
- Логистические показатели

**Источник:** СберИндекс API

### 4. Индекс мобильности

**Файл:** `mobility_index.xlsx` (~33 KB)

**Содержание:**
- Индекс мобильности населения
- Данные Росстат
- 128 населённых пунктов

**Источник:** Росстат

### 5. Справочники

**Файл:** `territory_id_pora.csv` (~24 KB)

**Содержание:**
- Маппинг ID территорий
- Соответствие между различными кодификаторами

**Источник:** Внутренний справочник проекта

## Климатические данные

Климатические данные (градусо-дни отопления) загружаются автоматически из **Open-Meteo API** при запуске скриптов.

**Источник:** https://open-meteo.com/

**Лицензия:** CC BY 4.0

**Требования:**
- Интернет подключение при первом запуске
- API ключ не требуется (бесплатный доступ)

## Структура данных после загрузки

После выполнения pipeline, в этой директории будут созданы промежуточные файлы:

```
data/
├── settlements_raw.csv              # Справочник НП (31 KB)
├── indicators_raw.csv               # POAD показатели (2.5 MB)
├── ndi_final.csv                    # NDI в CSV (22 KB)
├── attractiveness_v1.csv            # POAD scores (18 KB)
├── attractiveness_v2.csv            # С учётом доступности (29 KB)
├── accessibility_scores.csv         # Доступность (14 KB)
├── clusters.csv                     # Кластеры (16 KB)
├── distance_matrix.npy              # Матрица расстояний (131 KB)
└── ...

dashboard/
└── ndi_data.json                    # Финальные NDI данные для дашборда (123 KB)
```

⚠️ Эти файлы также **исключены из git** (см. `.gitignore`) и генерируются автоматически.

## Альтернатива

Если у вас нет доступа к исходным данным, вы можете:

1. **Использовать готовый dashboard** в `/dashboard/dashboard.html`
   - Все данные уже встроены
   - Не требует установки или загрузки дополнительных файлов

2. **Запросить тестовый датасет** у авторов проекта
   - Открыть [Issue](https://github.com/your-org/sberindex_dashboard/issues)
   - Указать цель использования данных
